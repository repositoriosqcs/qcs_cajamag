    2.7 Fact Historico Sub Prescrito¶

    Introducción¶

    El proceso ETL para Fact_HistoricoSubPrescrito permite extraer, transformar y cargar datos relacionados con la prescripción de subsidios de beneficiarios, consolidando la información en el Data Warehouse (DWH). Este flujo de trabajo se centra en los registros de prescripción de los últimos 18 meses, proporcionando detalles sobre beneficiarios, afiliación, y tipo de subsidio, y clasificando los subsidios como “Rural” o “No Rural” según corresponda. A lo largo del proceso, se generan identificadores únicos y se estandarizan nombres de columnas para facilitar análisis y reportes.

    Consulta de Prescripción de Subsidios por Beneficiario¶

    Esta consulta SQL obtiene información sobre la prescripción de subsidios en un período de 18 meses, agrupando datos de beneficiarios y sus detalles de afiliación. Se accede a la tabla principal subsi146 (representada como k1) y se realiza una unión interna con subsi22 (k2) para obtener datos de identificación como el sexo y un identificador único de afiliado (ID_AFILIADO). También se agrega una unión externa con subsi09 (k3) para clasificar el tipo de subsidio (tipsub) como "Rural" o "No Rural" basado en el campo tipsub.

    Se filtran registros donde estado es "P" y fecifz está dentro del rango de 18 meses desde la fecha actual. La consulta agrupa los resultados por periodo, codben, tipsub y ID_AFILIADO, proporcionando el valor total de prescripción por cada beneficiario en el rango de tiempo especificado.

    # Lista de querys
qr_structure = {
    "query": """
    select 
        fecifz as periodo, 
        SUM(valor) as valor_prescripcion, 
        k1.codben, 
        k2.sexo,
        CONCAT(k2.coddoc, k2.documento) as ID_AFILIADO,
        IF(k3.tipsub in ('A', 'G'), 'Rural', 'No Rural') as tipsub
    from subsidio.subsi146 as k1
    INNER JOIN (
        select codben, sexo, coddoc, documento 
        from subsidio.subsi22
    ) as k2
    ON k1.codben = k2.codben
    LEFT JOIN (
        select pergir, cedres, codben, tipsub 
        from subsidio.subsi09
    ) as k3
    ON k1.pergirpag = k3.pergir
    AND k1.cedres = k3.cedres
    AND k1.codben = k3.codben
    where estado = 'P' 
    AND fecifz BETWEEN DATE_SUB(CURDATE(), INTERVAL 18 MONTH) AND CURDATE()
    group by fecifz, k1.codben, tipsub, CONCAT(k2.coddoc, k2.documento);
    """
}

    Transformación de Columnas y Creación de Campos de Fecha Personalizados¶

    Este código realiza una serie de transformaciones en el DataFrame df, principalmente para estandarizar formatos y crear campos derivados basados en fechas. Primero, cambia el tipo de la columna codben a texto y convierte periodo en un objeto de fecha (datetime). Luego, agrega las columnas Año y Mes, derivadas del año y el mes de periodo, y crea una columna Mes_personalizado en formato de dos dígitos para asegurar la uniformidad.

    A partir de Año y Mes_personalizado, se crea una columna Periodo_2 en formato "AñoMes" y se elimina el campo periodo original, renombrando Periodo_2 a Periodo para estandarizar el nombre. La columna KeyPeriodos también se agrega como un duplicado de Periodo, posiblemente para facilitar búsquedas o uniones en etapas posteriores. Finalmente, la función convertir_columnas_mayusculas aplica el diccionario _names, renombrando codben a CODIGO_BENEFICIARIO y asegurando que los nombres de columnas estén en un formato consistente.

    df = df_structure['query']

# Cambiar el tipo de columna 'codben' a texto
df['codben'] = df['codben'].astype(str)

# Cambiar el tipo de columna 'periodo' a fecha
df['periodo'] = pd.to_datetime(df['periodo'])

# Agregar columnas Año y Mes
df['Año'] = df['periodo'].dt.year.astype(str)
df['Mes'] = df['periodo'].dt.month.astype(str)

# Agregar la columna Mes_personalizado (mes con formato de 2 dígitos)
df['Mes_personalizado'] = df['Mes'].apply(lambda x: x.zfill(2))

# Crear la columna Periodo_2 combinando Año y Mes_personalizado
df['Periodo_2'] = df['Año'] + df['Mes_personalizado']

# Eliminar la columna 'periodo' original
df = df.drop(columns=['periodo'])

# Renombrar la columna Periodo_2 a Periodo
