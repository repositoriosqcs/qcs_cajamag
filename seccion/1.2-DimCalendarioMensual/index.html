    1.2 Dim Calendario Mensual¶

    Intoducción¶

    Este proceso ETL tiene como finalidad generar y almacenar una tabla de calendario mensual en un Data Warehouse (DWH), facilitando la disponibilidad de un calendario estructurado y estandarizado para análisis de datos temporales. El flujo de trabajo crea un rango de fechas mensual desde hace 18 meses hasta el presente, extrayendo y estructurando campos de año, mes y primer día del mes. La información final se guarda en la tabla BD_Dim_Calendario del DWH para su uso en análisis de periodos mensuales.

    ETL¶

    Importación de librerías y funciones¶

    Este código configura el entorno de trabajo importando varias bibliotecas esenciales para la manipulación de datos y la conexión a bases de datos, como SQLAlchemy, Pandas y dateutil.relativedelta para cálculos de fechas. También gestiona rutas mediante sys y os para importar funciones personalizadas desde un archivo externo (Funciones.py). Una vez importadas, se ejecuta testfunciones(), presumiblemente una función de prueba que verifica el correcto funcionamiento o disponibilidad de las demás funciones importadas, confirmando que están listas para ser utilizadas en el flujo de trabajo.

    Configuración del logger¶

    Se configura el logger para registrar los eventos del proceso ETL en el archivo Calendario_Mensual.log con el nivel de detalle INFO, e inicia el registro indicando el comienzo del proceso.

    logger = setup_logger(log_filename='Calendario_Mensual.log', log_level=logging.INFO)
logger.info('COMIENZO ETL')

    2024-10-22 21:23:51,315 - INFO - COMIENZO ETL

    Creación de la tabla calendario¶

    Se genera un dataframe dfCalendar con un rango de fechas mensuales desde hace 18 meses hasta el presente. Luego se extraen los campos de año y mes, y se crea un campo Periodo en formato AñoMes. El proceso se registra en el log.

    #Creación tabla calendario
dfCalendar = pd.DataFrame( pd.date_range(date.today() - relativedelta(months = 18),date.today(), 
              freq='MS').tolist() , columns = ['PRIMER_DIA'] )
dfCalendar['ANIO'] = dfCalendar['PRIMER_DIA'].dt.year
dfCalendar['MES'] = dfCalendar['PRIMER_DIA'].dt.month
dfCalendar['PERIODO'] = dfCalendar['ANIO']*100 + dfCalendar['MES']
dfCalendar['PERIODO'] = dfCalendar['PERIODO'].astype(str)
logger.info('CREACION TABLA CALENDARIO')
# Nuevo orden de columnas
nuevo_orden = ['PERIODO', 'ANIO', 'MES','PRIMER_DIA']

# Reordenar usando reindex
dfCalendar = dfCalendar.reindex(columns=nuevo_orden).copy()
